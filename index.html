Orientador: Marco Dimas Gubitoso

Tema: Estudo de caso sobre o desenvolvimento de código limpo

Ano: 2016

Resumo: Este projeto tem por objetivo o estudo e utilização de técnicas para o desenvolvimento de código limpo. Para tal, será desenvolvida uma biblioteca, apelidada de SmartSampa, que facilitará o acesso a dados do sistema de ônibus da cidade. A prefeitura de São Paulo disponibiliza diversos dados a respeito da cidade, porém muitas vezes o acesso à eles não é facilmente automatizável. Como exemplo, podemos citar a plataforma GeoSampa. Será desenvolvido também um serviço web que irá utilizar a biblioteca para disponibilizar os dados via requisições HTTP. Na monografia, será mostrada a evolução do código da biblioteca e serão exploradas as diversas técnicas existentes para a confecção de código limpo.
